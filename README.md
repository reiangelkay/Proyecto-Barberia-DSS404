## Descripción del Proyecto
Bar4Beards es una plataforma web moderna (Single Page Application) diseñada para optimizar la gestión de barberías. Integra un sistema de reservas automatizado (con elección de barbero), una **cola de espera inteligente en tiempo real**, un sistema de feedback post-servicio y un catálogo interactivo para pedidos de bebidas con control de inventario.

## Stack Tecnológico y Arquitectura
Este proyecto utiliza una arquitectura desacoplada basada en el consumo de una API REST:
* **Frontend (Cliente):** React JS + Tailwind CSS
* **Backend (API REST):** PHP 
* **Base de Datos:** MySQL
* **Control de Versiones:** Git / GitHub
